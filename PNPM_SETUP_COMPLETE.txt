================================================================================
                      TROUBLESHOOTING
================================================================================

Problem: "dev.sh: command not found"
Solution: Run with bash explicitly
   bash dev.sh

Problem: "pnpm: command not found"
Solution: Make sure Node.js 20 is active
   export NVM_DIR="$HOME/.nvm"
   [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
   nvm use 20

Problem: Dependencies still installing slowly
Solution: Clear pnpm cache
   pnpm store prune
   pnpm install

Problem: Port 3000 already in use
Solution: Use a different port
   pnpm dev -- -p 3001
   # Open http://localhost:3001
